# Project-Multi-Modal-AI-System-for-Industrial-Quality-Assurance
This project develops an AI-based industrial quality inspection system. A YOLOv8 model detects defects such as cracks and scratches on steel surfaces, while an LLM generates maintenance reports and repair suggestions. The system is deployed through a Streamlit web app, combining computer vision and LLMs to automate quality assurance process
This project is a part of IIT Bombay's Season Of Code, spanning 8 weeks long. Work flow:Weeks 1 and 2 focus on building a strong foundation in Python programming and core Machine Learning concepts, including supervised and unsupervised learning, along with essential libraries such as NumPy and Pandas. Weeks 3 and 4 introduce Computer Vision and Large Language Models (LLMs), where participants learn how image-based AI models detect defects and how LLMs process and generate human-readable text. Weeks 5 and 6 concentrate on Data Engineering, Exploratory Data Analysis (EDA), preprocessing, and model training, involving data cleaning, feature preparation, and training a computer vision model to identify industrial defects such as cracks and scratches. Finally, Weeks 7 and 8 emphasize system integration and deployment, where the trained vision model is connected to an LLM to automatically generate maintenance reports from detected defects, and the complete end-to-end Quality Assurance system is deployed as an interactive Streamlit web application.
